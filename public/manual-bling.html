Manual de instalacao e autorizacao no Bling API v3

Etique Facil

Este manual orienta clientes a instalar o aplicativo Etique Facil pela Central de Extensoes do Bling, autorizar a integracao OAuth e concluir as configuracoes iniciais dentro do sistema.

Acessar Etique Facil

1. Sobre o aplicativo

O Etique Facil e um sistema para operacoes de estoque que importam lotes, conferem produtos por bipagem, imprimem etiquetas, criam produtos no Bling, lancam entradas de estoque e enviam transferencias entre depositos ou lojas.

Recursos usados no Bling

  • Produtos e cadastros relacionados.
  • Depositos e saldos de estoque.
  • Movimentacoes de entrada, saida e transferencia.

Quem deve instalar

A instalacao deve ser feita pelo usuario principal ou responsavel pela conta Bling da operacao.

2. Requisitos antes da instalacao

  1. Ter uma conta ativa no Bling com permissao para instalar aplicativos na Central de Extensoes.
  2. Ter uma conta de usuario principal no Etique Facil.
  3. Usar a mesma empresa/conta Bling que sera movimentada pelo Etique Facil.
  4. Permitir pop-ups e redirecionamentos do navegador durante a autorizacao, se o navegador solicitar.
O cliente nao precisa copiar Client ID, Client Secret, access token ou refresh token. A autorizacao e feita pelo fluxo seguro OAuth do Bling.

3. Passos no Bling

  1. Acesse sua conta Bling pelo navegador.
  2. Abra a Central de Extensoes.
  3. Use a barra de pesquisa para buscar Etique Facil.
  4. Ao localizar o aplicativo, clique em Instalar aplicativo.
  5. Na tela de autorizacao, confira as permissoes solicitadas.
  6. Clique em Autorizar para permitir que o Etique Facil acesse os recursos necessarios.
  7. Depois da autorizacao, o Bling redireciona automaticamente para o Etique Facil.

A autorizacao permite que o Etique Facil obtenha os tokens de acesso e se comunique com a conta Bling autorizada. A partir desse ponto, a etapa de instalacao no Bling esta concluida.

4. Passos no Etique Facil

Autorizar pela tela de sincronizacao

  1. Acesse o Etique Facil pelo endereco informado pela sua operacao.
  2. Entre com o usuario principal da conta.
  3. Abra Perfil.
  4. Clique em Sincronizacao.
  5. No painel Integracao Bling API, clique em Autorizar no Bling.
  6. Entre na conta Bling correta, se ainda nao estiver logado.
  7. Confirme a autorizacao no Bling.
  8. Ao retornar ao Etique Facil, confirme a mensagem de sucesso.

Quando a integracao estiver ativa, o sistema mostra a mensagem Bling autorizado para este usuario.

Depois da autorizacao

  1. Importe uma planilha de lote ou crie um lote avulso em Perfil > Entradas.
  2. Confira RZs ou pallets por bipagem na aba Conferencia.
  3. Imprima etiquetas dos produtos conferidos.
  4. Use Criar produtos no Bling para enviar novos SKUs quando necessario.
  5. Use Entrada Estoque Bling para registrar entradas de estoque.
  6. Use a aba Transferencias para montar remessas, conferir recebimento e enviar transferencias ao Bling.

5. Remover ou refazer a autorizacao

Use este processo somente quando precisar trocar a conta Bling autorizada ou corrigir uma autorizacao feita na conta errada.

  1. No Etique Facil, acesse Perfil > Sincronizacao.
  2. Clique em Remover autorizacao.
  3. Confirme a acao.
  4. Clique novamente em Autorizar no Bling e aprove o acesso na conta correta.

6. Solucao de problemas

O Bling nao autoriza

  • Confirme se esta logado na conta Bling correta.
  • Verifique se o usuario Bling tem permissao para instalar aplicativos.
  • Tente sair do Bling e iniciar a autorizacao novamente pelo Etique Facil.

O Etique Facil informa que o Bling nao esta autorizado

  • Acesse Perfil > Sincronizacao e confira o status da integracao.
  • Se necessario, remova a autorizacao e autorize novamente.
  • Confirme se a conta Bling autorizada e a conta usada pela operacao.

Produtos ou estoque nao enviam ao Bling

  • Confira se a integracao esta autorizada.
  • Verifique se os produtos possuem SKU, descricao e dados obrigatorios.
  • Para transferencias, confirme se os depositos de origem e destino existem no Bling.
